Forkphorus to exe with NW.js

  1. Use the packager to get a .html file for the project.
  2. Download NW.js's "Normal" version (not "SDK")
  3. Extract NW.js
  4. Move project.html (from the packager) into the extracted folder (the same folder where "nw.exe" is located)
  5. Create a file "package.json" in the same folder with the content: 
    {
  "name": "My project",
  "main": "project.html",
  "window": {
    "title": "My Project",
    "width": 480,
    "height": 360
  }
}
    Sections that are underlined may be changed.
  6. Open nw.exe
  7. You're done!

The same process can be done for Mac OS and Linux.
